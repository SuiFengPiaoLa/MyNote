## 一、obsidian同步到github[教程]([Obsidian文件同步方案：Git - 彻底告别文件冲突和丢失_哔哩哔哩_bilibili](https://www.bilibili.com/video/BV1JUtuzUECZ/?spm_id_from=333.337.search-card.all.click))

> [!info] 前置条件
 >1.下载[git官网](https://git-scm.com/)和安装git。
 >2.下载[obdidian官网](https://obsidian.md/download)))和安装obsidian软件。
 >3.注册和登录github([GitHub](https://github.com/))

### 1.从github/gitee仓库克隆本地电脑

 **github：cmd：git clone https://github.com/SuiFengPiaoLa/MyNote
 
 **gitee:  cmd：git clone https://gitee.com/suifengpiaola/mynote
### 2.从本地电脑到github/gitee

 **github：cmd：git clone https://github.com/SuiFengPiaoLa/MyNote
 
 **gitee:  cmd：git clone https://gitee.com/suifengpiaola/mynote

**第二步：配置 Dataview**

1. 设置 → Dataview → 开启「内联查询」和「JavaScript 查询」
2. 模板中的 `dataview` 代码块会自动渲染为动态表格

**第三步：开始使用**

1. `Ctrl+P` 打开命令面板
2. 输入 `Templater` → 选择对应模板
3. 填写笔记内容，保存即可
